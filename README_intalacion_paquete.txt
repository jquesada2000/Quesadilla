

Para instalar el paquete Quesadilla desde Github, con las dos Vignettes correspondientes al 
codigo del Libro "Bioestadística Básica y Avanzada con R" de la Editorial Diaz de Santos, 2024, ejecutar:

Instalar devtools:

install.packages("devtools")



Instalar Quesadilla:

devtools::install_github("jquesada2000/Quesadilla", build_vignettes = TRUE,force = TRUE)
